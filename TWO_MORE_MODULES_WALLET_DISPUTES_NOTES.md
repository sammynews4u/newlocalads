# Two More Local Ads Modules Added

This update adds two further public-facing product modules to the Local Ads codebase.

## 1. Payments & Wallet Operations Module

Route: `/wallet-operations`

Purpose:
- Explain advertiser wallet funding.
- Clarify publisher earnings and available/pending balances.
- Define transaction ledger expectations.
- Describe withdrawal review and settlement controls.

This module supports future dashboard implementation for campaign spend control, publisher payout transparency, platform revenue-share tracking and admin reconciliation.

## 2. Disputes & Refund Management Module

Route: `/disputes`

Purpose:
- Explain invalid-click complaints.
- Define publisher payout appeals.
- Structure refund requests.
- Set expectations for evidence, case status, admin decisions and audit history.

This module supports future dashboard implementation for case management, complaint tracking, refund approval, payout corrections and policy appeals.
